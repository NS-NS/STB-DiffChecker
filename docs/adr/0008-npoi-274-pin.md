# 0008: NPOIを2.7.4に固定（ライセンス理由）

- ステータス: 採用（2026-07）

## 文脈
Excel出力に使用するNPOIの新しいバージョンにはライセンス上の問題がある:
- 2.8.0以降: 商用利用に保守料を求めるEULAに変更された
- 2.7.5以降: NSax(LGPL)を含むようになった

本ツールはMITライセンスで、企業内利用が主な想定のため、
LGPLや有償EULAの混入は避けたい。

## 決定
- NPOIは**2.7.4に固定**する(純粋にApache-2.0系の依存で構成される最後の版)
- 推移的依存の脆弱性は、修正版パッケージの明示的なPackageReference追加で上書きする
  (NPOI本体を上げずに脆弱性だけ塞ぐ)
- OSSライセンス表示(サードパーティ通知)を同梱する

## 再検討の条件
- NPOIのライセンス方針が変わった場合
- 2.7.4系で塞げない脆弱性が出た場合(代替: ClosedXML等への移行、またはExcel出力の
  CSV代替を検討)

# 0009: ユーザー設定はレジストリでなくJSONファイル

- ステータス: 採用（2026-07。旧レジストリ保存を置き換え）

## 文脈
元は最後に使ったフォルダをレジストリ(HKCU\NS-NS\STB-DiffChecker)に保存していた。
アンインストール後にゴミが残る、内容が見えにくい、設定項目を増やしにくい、
という難点があった。

## 決定
- `%APPDATA%\STB-DiffChecker\settings.json` に保存する(DiffCheckerLib.WPF.UserSettings)
- 旧レジストリ値は初回起動時に自動で読み継ぐ(移行後もレジストリは消さない。実害なし)
- 読み書きの失敗は動作に影響させない(握りつぶしてデフォルト動作)

## 影響
- 設定項目を増やす場合はUserSettings.Settingsクラスにプロパティを足すだけ
- Web版はブラウザ内で完結するためこの仕組みを使わない
